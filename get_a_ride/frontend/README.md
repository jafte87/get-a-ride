# 🚕 Ride Price Calculator

A web app to compare ride prices between Uber, inDrive, and DiDi in Egypt.

## 🛠️ Tech Stack

- React.js
- Leaflet / React-Leaflet
- CSS

## ✨ Features

- Select two points on the map
- Calculate distance between them
- Compare prices across 3 apps
- Responsive design

## 🚀 Getting Started

### Prerequisites
- Node.js
- npm

### Installation

```bash
git clone https://github.com/jafte87/your-repo-name
cd frontend
npm install
npm run dev
```
